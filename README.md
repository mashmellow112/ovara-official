# Novara - Premium African & Global Cinema Streaming Platform

![Novara Logo](images/favicon-gold.svg)

> Uganda's #1 streaming platform for local African movies and international content

## Overview

Novara is a modern, responsive web application serving as a coming soon landing page for a premium streaming service in Uganda. The platform showcases a curated selection of African and global cinema, offering users the ability to join a waitlist for early bird pricing (50% off).

## Features

### 🎬 Content Library
- **Local African Content** - Ugandan and East African films
- **Nollywood** - Nigerian movie industry selections
- **Bollywood** - Indian cinema
- **Hollywood** - International movies

### ✨ Platform Highlights
- **4K Ultra HD** streaming quality
- **Multi-Device Support** - Watch on TV, mobile, tablet, and desktop
- **Offline Downloads** - Download content for offline viewing
- **Fast Local Servers** - Optimized for Uganda and East Africa
- **Local Language Support** - Content in multiple African languages
- **Regional Content Library** - Curated African content

### 🔧 Technical Features
- **Progressive Web App (PWA)** - Installable on mobile devices
- **Responsive Design** - Works on all screen sizes
- **SEO Optimized** - Full meta tags and Open Graph support
- **Firebase Integration** - Waitlist signup with Firestore
- **Modern UI** - Gold-themed dark interface with smooth animations
